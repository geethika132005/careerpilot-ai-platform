# CareerPilot AI

CareerPilot AI is a comprehensive job application tracking platform designed to help students, graduates, and job seekers manage their placement journey efficiently. The platform provides a centralized workspace for tracking applications, monitoring progress, preparing for interviews, managing resumes, and organizing professional contacts.

## Live Demo

https://careerpilot-ai-platform.vercel.app

---

## Overview

CareerPilot AI simplifies the job search process by providing a structured platform where users can manage applications, track progress through various recruitment stages, prepare for interviews, and maintain career-related resources in one place.

---

## Key Features

### Job Application Management

* Add and manage job applications
* Track application status throughout the recruitment process
* Organize opportunities using a Kanban-style workflow
* Search and filter applications efficiently

### Application Pipeline Tracking

Supports multiple recruitment stages:

* Applied
* In Touch
* Technical Round
* HR Interview
* Offer
* Rejected

### Analytics Dashboard

* Monitor total applications
* Track active opportunities
* View offer statistics
* Analyze rejection trends
* Measure application progress

### Interview Preparation Module

* Technical interview preparation resources
* HR interview preparation materials
* Common interview questions and guidance
* Structured interview readiness support

### Resume Management

* Store and organize resumes
* Manage multiple resume versions
* Centralized access to career documents

### Contact Management

* Maintain recruiter and company contacts
* Organize professional networking information
* Track communication details

### Calendar Management

* Schedule interviews and follow-ups
* Track important deadlines
* Manage placement-related events

### User Experience

* Responsive interface
* Modern dashboard design
* Intuitive navigation
* Real-time application management

## Installation

### Prerequisites

* Node.js
* npm

### Setup

Clone the repository:

```bash
git clone https://github.com/geethika132005/careerpilot-ai-platform.git
```

Navigate to the project directory:

```bash
cd careerpilot-ai-platform
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm run dev
```

Open your browser and access:

```text
http://localhost:5173
```
